# 价格分布图显示问题修复报告

## 🐛 问题描述

用户反馈：**价格分布图显示有问题，价格会超出显示范围，貌似还只有一个区间数据**

### 问题现象
1. **Y 轴范围不当**: 图表 Y 轴没有自动调整，导致高价值物品的价格点可能超出可视范围
2. **区间设置不合理**: 使用固定的价格区间（0-100, 100-200, ..., 2000+），无法适应不同价位的物品
3. **数据展示不全**: 高价值物品（如哮喘吸入器 65k-112k）所有数据都落在"2000+"区间，实际只显示一个柱子

---

## 🔍 问题分析

### 原代码逻辑

```javascript
// ❌ 问题代码：固定的价格区间
const priceRanges = [0, 100, 200, 500, 1000, 2000, 5000];
const rangeLabels = ['0-100', '100-200', '200-500', '500-1000', '1000-2000', '2000+'];
const rangeCounts = new Array(rangeLabels.length).fill(0);

prices.forEach(price => {
    for (let i = 0; i < priceRanges.length - 1; i++) {
        if (price >= priceRanges[i] && price < priceRanges[i + 1]) {
            rangeCounts[i]++;
            break;
        }
    }
    if (price >= priceRanges[priceRanges.length - 1]) {
        rangeCounts[rangeCounts.length - 1]++;
    }
});
```

### 问题根源

| 物品 | 价格范围 | 落入区间 | 显示结果 |
|------|---------|---------|---------|
| 哮喘吸入器 | 65,984 - 112,089 | 全部 > 5000 | 只有"2000+"有数据 ❌ |
| 野战急救包 | 8,500 - 13,500 | 全部 > 5000 | 只有"2000+"有数据 ❌ |
| 低级材料 | 50 - 120 | 50-100, 100-200 | 正常显示 ✅ |

**结论**: 固定区间只适用于低价物品，无法适配全价位段

## ✅ 修复方案

### 核心改进

#### 1. **动态计算价格区间**
根据实际数据的最小值、最大值和数据量，自动计算合理的区间划分。

```javascript
// ✅ 修复后：动态区间计算
const minPrice = Math.min(...prices);
const maxPrice = Math.max(...prices);
const priceRange = maxPrice - minPrice;

// 根据数据量确定区间数量
let numRanges = 5;
if (prices.length > 20) numRanges = 7;
if (prices.length > 30) numRanges = 8;

// 计算区间大小并圆整到美观的值
const rangeSize = roundToNiceNumber(Math.ceil(priceRange / numRanges));

// 计算实际范围
const actualMin = Math.floor(minPrice / rangeSize) * rangeSize;
const actualMax = Math.ceil(maxPrice / rangeSize) * rangeSize;
const actualRanges = Math.ceil((actualMax - actualMin) / rangeSize);
```

#### 2. **智能圆整算法**

```javascript
/**
 * 将数字圆整到美观的值（如 100, 500, 1000 等）
 * 示例：
 * - 783 → 1000
 * - 456 → 500
 * - 1234 → 2000
 * - 9876 → 10000
 */
function roundToNiceNumber(num) {
    if (num <= 0) return 100;
    
    const magnitude = Math.pow(10, Math.floor(Math.log10(num)));
    const normalized = num / magnitude;
    
    let niceNormalized;
    if (normalized <= 1) niceNormalized = 1;
    else if (normalized <= 2) niceNormalized = 2;
    else if (normalized <= 5) niceNormalized = 5;
    else niceNormalized = 10;
    
    return niceNormalized * magnitude;
}
```

#### 3. **Y 轴自动缩放**

```javascript
// ✅ 添加 autorange: true 让 Plotly 自动调整 Y 轴范围
Plotly.newPlot('priceChart', [priceTrace], {
    title: `${itemName} 价格走势`,
    xaxis: { title: '日期' },
    yaxis: { 
        title: '价格 (哈弗币)',
        autorange: true  // ← 自动调整
    },
    // ...
});
```

#### 4. **增强的柱状图样式**

```javascript
const distTrace = {
    x: rangeLabels,
    y: rangeCounts,
    type: 'bar',
    marker: { 
        color: '#27ae60',
        line: {
            color: '#1e8449',
            width: 1
        }
    },
    text: rangeCounts,      // ← 显示数值
    textposition: 'auto',   // ← 自动定位
    hoverinfo: 'x+y'        // ← 悬停提示
};
```

#### 5. **数字格式化**

```javascript
/**
 * 格式化数字（添加千位分隔符）
 * 示例：
 * - 65984 → "65,984"
 * - 112089 → "112,089"
 */
function formatNumber(num) {
    return num.toString().replace(/\B(?=(\d{3})+(?!\d))/g, ",");
}
```

## 💡 边界情况处理

### 1. **数据量极少**（< 5 条）
```javascript
// 默认 5 个区间，确保基本可读性
let numRanges = 5;
```

### 2. **数据量适中**（5-20 条）
```javascript
// 7 个区间，平衡细节与简洁
if (prices.length > 20) numRanges = 7;
```

### 3. **数据量丰富**（> 30 条）
```javascript
// 8 个区间，展示更多分布细节
if (prices.length > 30) numRanges = 8;
```

### 4. **价格完全相同**（rangeSize = 0）
```javascript
// roundToNiceNumber 会返回默认值 100
if (num <= 0) return 100;
```

### 5. **最大值边界**
```javascript
// 确保最大值也能落入最后一个区间
if (index >= actualRanges) {
    rangeCounts[actualRanges - 1]++;
}
```
