
## 🎯 技术实现细节

### 1. Bootstrap 栅格系统

#### 水平布局（统计信息）
```html
<div class="row justify-content-center g-3">
    <div class="col-auto">...</div>     <!-- 自适应宽度 -->
    <div class="col-auto">...</div>     <!-- 自适应宽度 -->
</div>
```

**关键类说明**:
- `row`: 创建弹性容器
- `justify-content-center`: 水平居中
- `g-3`: 栅格间距（1rem）
- `col-auto`: 根据内容自动调整宽度

#### 垂直布局（图表）
```html
<div class="row mb-4">
    <div class="col-12">...</div>       <!-- 占满整行（12 列） -->
</div>
```

**关键类说明**:
- `col-12`: 占满整行（Bootstrap 栅格共 12 列）
- `mb-4`: 底部外边距（1.5rem），增加图表间距

### 2. 响应式适配

```css
/* 小屏幕时自动调整 */
@media (max-width: 768px) {
    .stats-badge {
        padding: 0.4rem 1rem;
        font-size: 0.9rem;
        min-width: 240px;
    }
}
```

**移动端表现**:
- 统计徽章可能自动换行（取决于屏幕宽度）
- 图表保持垂直布局，宽度自适应

## 💡 设计原则

### 1. 内容决定形式
- **统计信息**: 简短文本 → 水平分布更高效
- **图表数据**: 复杂可视化 → 垂直分布更清晰

### 2. 空间优化
- 统计徽章占用空间小 → 并排显示
- 图表需要大视野 → 独占整行

### 3. 视觉层次
```
第一层：标题（Hero 区域）
第二层：统计信息（关键指标）
第三层：物品卡片网格
第四层：详细分析图表（弹窗）
```

### 4. 响应式思维
- 桌面端：最优布局
- 移动端：自动降级（换行、缩放）
- 始终保持可读性和可用性
