# 模板组件化重构说明文档

## 📋 重构概述

已将原来的单文件 HTML（784 行）重构为**模块化、组件化**的模板结构，提升可维护性和可扩展性。

---

## 🏗️ 新的目录结构

```
templates/
├── base.html                 # 基础模板（公共布局、CSS/JS 依赖）
├── index.html                # 首页（继承 base.html）
├── layouts/                  # 布局片段目录
│   └── _header.html          # 头部导航 + 英雄区域
└── components/               # 组件目录
    ├── _item_card.html       # 物品卡片组件
    ├── _chart_modal.html     # 图表弹窗组件
    └── _search_bar.html      # 搜索栏组件
```

---

## 🎯 各模块职责

### 1. **base.html** - 基础模板
- **作用**: 定义整体页面框架
- **包含内容**:
  - HTML `<head>` 部分
  - CSS 样式定义
  - 导航栏占位符 (`{% include 'layouts/_header.html' %}`)
  - 主内容区域 (`{% block content %}`)
  - 模态框占位符 (`{% include 'components/_chart_modal.html' %}`)
  - 底部信息
  - JS 依赖引入（Bootstrap、Plotly.js）
  - 额外 JS 块 (`{% block extra_js %}`)

### 2. **index.html** - 首页
- **继承**: `{% extends "base.html" %}`
- **作用**: 展示物品列表和交互逻辑
- **包含内容**:
  - 搜索区域 (`{% include 'components/_search_bar.html' %}`)
  - 物品列表容器
  - 分页组件
  - JavaScript 交互逻辑（加载数据、图表绘制等）

### 3. **layouts/_header.html** - 头部布局
- **作用**: 网站头部统一样式
- **包含**:
  - 导航栏（品牌 Logo、分类菜单）
  - 英雄区域（统计信息展示）

### 4. **components/_item_card.html** - 物品卡片
- **作用**: 单个物品的展示卡片
- **特性**:
  - 使用 Jinja2 模板语法
  - 支持图片懒加载和错误处理
  - 点击显示分析图表

### 5. **components/_chart_modal.html** - 图表模态框
- **作用**: 物品详情分析弹窗
- **包含**:
  - Bootstrap Modal 结构
  - 动态内容加载区域

### 6. **components/_search_bar.html** - 搜索栏
- **作用**: 搜索和筛选功能
- **包含**:
  - 搜索输入框
  - 刷新按钮
  - 加载动画

## 🔧 技术亮点

### 1. **模板继承系统**
```jinja2
{# base.html #}
<html>
<head>...</head>
<body>
    {% block content %}{% endblock %}
    {% block extra_js %}{% endblock %}
</body>
</html>

{# index.html #}
{% extends "base.html" %}
{% block content %}
    <!-- 页面内容 -->
{% endblock %}
{% block extra_js %}
    <script>...</script>
{% endblock %}
```

### 2. **组件复用机制**
```jinja2
{% include 'components/_item_card.html' %}
{% include 'layouts/_header.html' %}
```

### 3. **自定义 Jinja2 过滤器**
```python
# app/__init__.py
@app.template_filter('format_time')
def format_time_filter(time_string):
    """格式化时间字符串"""
    return datetime.fromisoformat(time_string).strftime('%Y-%m-%d %H:%M:%S')
```

使用方式：
```jinja2
{{ item.crawl_time | format_time }}
```

## 📝 使用指南

### 添加新页面

1. **创建页面模板**（继承 base.html）
```jinja2
{# templates/new_page.html #}
{% extends "base.html" %}

{% block title %}新页面标题{% endblock %}

{% block content %}
    <div class="container">
        <!-- 页面内容 -->
    </div>
{% endblock %}

{% block extra_js %}
    <script>
        // 页面专属 JS
    </script>
{% endblock %}
```

2. **添加路由**
```python
# app/routes/main_routes.py
@bp.route('/new-page')
def new_page():
    return render_template('new_page.html')
```

### 添加新组件

1. **创建组件文件**
```jinja2
{# templates/components/_new_component.html #}
<div class="new-component">
    <!-- 组件内容 -->
</div>
```

2. **在页面中使用**
```jinja2
{% include 'components/_new_component.html' %}
```

## 🎨 样式定制

所有全局样式定义在 `base.html` 的 `<style>` 标签中，可通过 CSS 变量轻松定制：

```css
:root {
    --primary-color: #2c3e50;      /* 主色调 */
    --secondary-color: #3498db;    /* 次级色 */
    --accent-color: #e74c3c;       /* 强调色 */
    --background-color: #ecf0f1;   /* 背景色 */
    --card-bg: #ffffff;            /* 卡片背景 */
}
```

## 🐛 常见问题

### Q1: 如何修改导航栏？
A: 编辑 `templates/layouts/_header.html`

### Q2: 如何添加新的 CSS 样式？
A: 在 `base.html` 的 `<style>` 标签中添加，或创建独立 CSS 文件后在 `<head>` 中引入

### Q3: 如何在其他页面使用图表功能？
A: 
1. 在页面中包含 `{% include 'components/_chart_modal.html' %}`
2. 确保加载了 Plotly.js: `<script src="{{ url_for('static', filename='js/plotly.min.js') }}"></script>`
3. 调用 `drawAnalysisCharts()` 函数

### Q4: 如何优化大数据量加载？
A: 
- 使用 API 分页（已实现）
- 启用 Redis 缓存（已实现）
- 考虑添加懒加载（滚动加载更多）
