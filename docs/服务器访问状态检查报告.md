### `#` 的含义
- `#` 是 **URL 锚点标记**（Anchor）
- 用于指定页面内的某个位置
- 例如：`http://example.com/page.html#section1`

## 🔧 如果仍然无法访问

### 可能原因及解决方案

#### 1. **云服务器安全组未开放端口**
**检查方法**:
- 登录云服务器控制台（阿里云/腾讯云/华为云等）
- 找到"安全组"或"防火墙"设置
- 检查是否有入站规则允许 TCP 5000 端口

**解决方法**:
```
添加入站规则：
- 协议类型：TCP
- 端口范围：5000
- 授权对象：0.0.0.0/0（允许所有 IP）
- 优先级：1
```

#### 2. **系统防火墙阻止连接**
**检查方法**:
```bash
# 查看防火墙状态
sudo ufw status
# 或
sudo firewall-cmd --state
```

**解决方法**:
```bash
# Ubuntu/Debian
sudo ufw allow 5000/tcp
sudo ufw reload

# CentOS/RHEL
sudo firewall-cmd --permanent --add-port=5000/tcp
sudo firewall-cmd --reload
```

#### 3. **SELinux 限制**（CentOS/RHEL）
**检查方法**:
```bash
getenforce
```

**临时解决**:
```bash
sudo setenforce 0
```

**永久解决**:
```bash
# 修改 /etc/selinux/config
SELINUX=permissive
```

#### 4. **路由器端口转发**（如使用家庭宽带）
**配置方法**:
- 登录路由器管理界面
- 找到"端口转发"或"虚拟服务器"
- 添加规则：
  - 外部端口：5000
  - 内部 IP: 服务器的局域网 IP（如 192.168.1.100）
  - 内部端口：5000
  - 协议：TCP

#### 5. **ISP 封锁端口**
**检测方法**:
```bash
# 在服务器上测试
curl http://82.156.116.206:5000
```

**解决方法**:
- 更换其他端口（如 8080、8888）
- 联系 ISP 确认端口政策
